# Bit-Operation
Битовые операции (НЕ, И, ИЛИ, XOR, сдвиги бит)


## Операнды в приоритетном порядке:

* `~` - НЕ
* `&` - И (проверка на включеные биты в числе)
* `&~` - НЕ И (выключение битов)
* `|` - ИЛИ (включение битов)
* `^` - XOR (исключающее ИЛИ)

## Таблицы соответствия:
### НЕ (~)
| **X** | **НЕ** |    
|:-----:|:----:  |
